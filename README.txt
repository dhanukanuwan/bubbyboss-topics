== Description ==

This plugin helps you to gather all content of your BuddyBoss site into one place. Plugin will create a new post type called Topics.
When you visit a topic page( let's say the topic's title is Marketing), it looks for posts,courses,images and videos with a category as same as the topic name( Marketing).
It will display all items found with a pagination. You also will be able to filter content by the content type.
